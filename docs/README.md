# 可源生物科技官方网站

这是一个可源生物科技的官方网站单机测试版，包含站长可编辑功能，支持随时更新图片、新闻以及公司信息。

## 网站结构

```
├── index.html          # 首页
├── about.html          # 公司信息页面
├── products.html       # 产品服务页面
├── news.html           # 新闻动态页面
├── contact.html        # 联系我们页面
├── admin.html          # 站长后台页面
├── styles.css          # 样式文件
├── script.js           # 前端交互逻辑
├── admin.js            # 后台管理功能
├── data.js             # 数据存储和管理
└── README.md           # 说明文档
```

## 功能特点

### 前端网站
- 响应式设计，适配不同设备
- 轮播图展示
- 公司信息展示
- 产品服务展示
- 新闻动态展示
- 联系信息展示

### 后台管理
- 管理员登录/退出功能
- 轮播图管理（添加、编辑、删除）
- 公司信息编辑
- 新闻管理（添加、编辑、删除）
- 产品管理（添加、编辑、删除）
- 联系信息编辑

## 技术实现

- **前端框架**：纯HTML/CSS/JavaScript
- **数据存储**：localStorage（本地存储，适合单机测试）

## 如何运行

### 方法一：直接在浏览器中打开

1. 找到项目文件夹
2. 直接双击 `index.html` 文件
3. 在浏览器中查看网站

### 方法二：使用本地HTTP服务器

推荐使用HTTP服务器来运行，避免浏览器的跨域限制，特别是在使用后台管理功能时。

#### 使用Python

```bash
# Python 3
python -m http.server 8000

# 然后在浏览器中访问 http://localhost:8000
```

#### 使用Node.js

```bash
# 安装http-server（如果尚未安装）
npm install -g http-server

# 启动服务器
http-server -p 8000

# 然后在浏览器中访问 http://localhost:8000
```

## 数据管理

所有数据都存储在浏览器的localStorage中，刷新页面后数据会保留。如果需要重置数据，可以：

1. 打开浏览器开发者工具（按F12）
2. 切换到"Application"或"Storage"标签
3. 找到"Local Storage"
4. 删除"websiteData"条目
5. 刷新页面，数据会恢复为默认值

## 自定义修改

### 修改默认数据

可以编辑 `data.js` 文件中的 `defaultData` 对象，修改网站的默认数据。

### 修改样式

可以编辑 `styles.css` 文件，自定义网站的样式。

### 添加新功能

可以编辑 `script.js` 和 `admin.js` 文件，添加新的功能模块。

1. 本网站为单机测试版，数据仅存储在本地浏览器中，不适合生产环境使用
2. 生产环境建议使用服务器端数据库存储数据
3. 后台管理没有做权限控制，仅用于测试
4. 图片使用了占位图服务，实际使用时请替换为真实图片
